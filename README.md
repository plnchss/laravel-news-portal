#  Laravel News Portal

Полнофункциональный новостной портал с развитой системой управления контентом (CMS). Проект ориентирован на удобное администрирование новостей и активное взаимодействие с аудиторией.

##  Основные возможности
* **Администрирование:** Удобная панель управления для постинга новостей, управления категориями и модерации комментариев.
* **Система комментариев:** Пользователи могут оставлять отзывы к статьям (реализовано через API).
* **Медиа-менеджмент:** Поддержка загрузки и обработки изображений для статей.
* **Личный кабинет:** Персонализированный профиль пользователя с историей активности.
* **Безопасность:** Реализованы политики доступа (**Policies**) — права на редактирование контента строго ограничены ролью администратора.
* **Уведомления:** Настроенная система почтовых оповещений через **Mail Mailer** (о событиях на портале, ответах на комментарии).

##  Технологический стек
* **Framework:** Laravel 11.x
* **Frontend:** Bootstrap (адаптивная верстка)
* **API:** Разработан RESTful API для взаимодействия с контентом.
* **Infrastructure:** Docker (контейнеризация), Mailer (SMTP интеграция)
